# Vibe Mode

Parse vague requests into structured interpretation.

## Workflow

1. Read the request and identify the likely goal.
2. Read repository instructions, `AGENTS.md`, rules, and relevant files.
3. Gather enough context before deciding.
4. If uncertain, ask Fahad one short question at a time. Do not guess.
5. If confident, state the interpretation and plan to execute.

Prefer concrete next actions over broad commentary.
